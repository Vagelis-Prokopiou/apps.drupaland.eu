Peribiblicum app

ServiceApp is an Android application (written in Java), that helps you simplify and automate the management of the periodic maintenance of your vehicle.

It is available through Google Play at the following url:

https://play.google.com/store/apps/details?id=com.github.vaggos.serviceapp

You can also visit the GitHub repository for more info at the following url:

https://github.com/Vaggos/ServiceApp_Android

There also is a Python CLI edition of the same app which can be found at:

https://github.com/Vaggos/serviceApp

Enjoy!
